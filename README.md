# Estoa

Estoa is a framework for building peer to peer, anonymous and collaborative applications.

## Features

- End-to-end encrypted communication between users.
- Fast local storage.
- P2P communication through hole-punching with QUIC.

## CRDTs

- G-Counter
- PN-Counter
- LWW-Register
- OR-Set
- RGA
- G-Set
- OR-Map

## Computed Datatypes

- Text
- Number
- List
